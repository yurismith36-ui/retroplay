Gere o bloco do seu novo jogo

Você preenche os campos e o RetroPlay cria o trecho certo para colocar no arquivo dados/games.json.

1. PREENCHA OS DADOS
2. COPIE O RESULTADO
Preencha o formulário para ver a pasta correta.
Atenção: esta página gera o cadastro, mas não modifica o GitHub automaticamente.
3. PASSO A PASSO PARA PUBLICAR
  1. Crie a pasta que aparece no quadro acima.
  2. Dentro dela, crie as pastas rom e imagens.
  3. Coloque a ROM na pasta rom.
  4. Coloque capa.jpg, banner.jpg e logo.png em imagens.
  5. No GitHub, abra dados/games.json e clique no lápis para editar.
  6. Depois do último jogo, coloque uma vírgula e cole o novo bloco.
  7. Salve em Commit changes.
Use apenas arquivos que você tenha autorização para hospedar. RetroPlay 1.0

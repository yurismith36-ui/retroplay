INSTALAÇÃO NO GITHUB
1. Extraia este ZIP no computador.
2. Abra seu repositório retroplay.
3. Para instalar somente o V7, remova o conteúdo antigo do repositório.
4. Envie TODO o conteúdo que está dentro da pasta RetroPlay-V7-Completo.
5. index.html precisa ficar na raiz do repositório.
6. Confirme as alterações.
7. Aguarde o GitHub Pages concluir a implantação.
8. Abra o site e pressione Ctrl+F5.

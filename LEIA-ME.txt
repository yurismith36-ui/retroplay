RETROPLAY — CONSOLE CORNER COM 3 FUNDOS ALEATÓRIOS

ALTERAÇÕES
- Três cenários:
  1. Locadora moderna
  2. Galáxia
  3. Locadora vintage
- Um cenário diferente é escolhido sempre que a página inicial é carregada.
- O script evita repetir imediatamente o mesmo cenário.
- Borda branca da capa removida.
- Nova borda preta com sombra.
- Área de informações com fundo escuro, conforme as imagens aprovadas.
- Mantém o jogo em destaque mudando diariamente.
- Responsivo para computador e celular.

ARQUIVOS
- index.html
- css/style.css
- js/app.js
- js/console-corner.js
- imagens/backgrounds/locadora-moderna.jpg
- imagens/backgrounds/galaxia.jpg
- imagens/backgrounds/locadora-vintage.jpg

COMO INSTALAR
1. Extraia este ZIP.
2. Abra o repositório RetroPlay no GitHub.
3. Clique em Add file > Upload files.
4. Envie index.html e as pastas css, js e imagens.
5. Aceite substituir os arquivos existentes.
6. Clique em Commit changes.
7. Aguarde de 1 a 3 minutos.
8. Abra a página inicial e pressione Ctrl + F5.

NÃO ALTERA
- ROMs
- dados/games.json
- capas individuais dos jogos

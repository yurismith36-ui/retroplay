# RetroPlay V4 — Estilo Steam

## Recursos incluídos

- Visual inspirado na Steam
- Banner com imagem grande à esquerda e informações à direita
- Avaliação por estrelas
- Pesquisa
- Filtros por console
- Favoritos salvos no navegador
- Janela de detalhes do jogo
- Galeria de imagens simulada
- Jogos relacionados
- Layout responsivo para computador e celular

## Como publicar

1. Extraia o ZIP.
2. Abra seu repositório RetroPlay no GitHub.
3. Clique em **Adicionar arquivo > Carregar arquivos**.
4. Envie:
   - `index.html`
   - `README.md`
   - `INSTRUCOES.txt`
   - pasta `css`
   - pasta `js`
   - pasta `jogos`
5. Confirme as alterações.
6. Aguarde cerca de um minuto.
7. Abra o site e pressione `Ctrl + F5`.

## Aviso

Use somente jogos próprios, homebrews, domínio público ou conteúdos para os quais você tenha autorização.
